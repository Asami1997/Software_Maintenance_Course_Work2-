README FOR SOFTWARE MAINTENANCE G52SWM CW2

Instructions

1.IMPORTANT: make sure to pick skin in the mapviewer before pressing play, otherwise this is unexpected behaviour.

Drag and drop:

1- click and hold on axe or boat 
2- drag to desired location 
3- drop at location where squares are not red.

Skin selection:
1-click on any of the desired skin and the game will update on that 

Zoom:
1- click on “+” or “ - ” for desired functionality 

After the user is finished with all the settings you would like to use, simply click the “Play” button and the game will start promptly.
